# Ashwin PI website redesign

This package applies the redesigned dark theme to the whole website.

## What changed

- The navbar is now a very small floating sidebar overlay.
- It no longer reserves space in the page layout, so the content gets more focus.
- On desktop/laptop screens, the navbar floats on the right side.
- The full page name appears only when hovering or focusing a nav button.
- On phones/tablets, the navbar becomes a compact floating top pill with horizontal shortcuts.
- The active page is highlighted automatically by `script.js`.
